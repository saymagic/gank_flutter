## Getting Started

#### 1. [Setup Flutter](https://flutter.io/setup/)
#### 2. Clone the repo
```sh
$ git clone https://github.com/saymagic/gank_flutter.git
$ cd gank_flutter/
```

## Run app

You need latest version of Flutter installed in your system for build and run the project.

```sh
$ flutter run
```
